# SnapQR

SnapQR is a Dynamic QR Image Hosting Platform built using Python and Flask.

## Features

- Upload Images
- Generate QR Codes
- Permanent Image Links
- Download QR
- Scan Analytics (Upcoming)
- Cloud Storage (Upcoming)

## Tech Stack

- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript
