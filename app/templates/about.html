About SnapQR

SnapQR is a modern Flask-based web application that lets users upload an image and instantly generate a QR Code linked to that image.

🚀 Our Mission

Our mission is to make image sharing fast, simple and accessible using QR technology. Upload once and share anywhere.

⚡ Why SnapQR?

  • ✔ Instant QR Generation
  • ✔ Secure Image Upload
  • ✔ Permanent Image Links
  • ✔ Clean & Responsive Design

Built using Flask + Tailwind CSS

SnapQR demonstrates image upload, QR generation, responsive UI, Cloud integration and clean Flask architecture.
